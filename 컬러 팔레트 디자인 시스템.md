## 컬러 사용 가이드

### Primary — `#F4845F` 웜 오렌지
- **사용 위치**: 메인 CTA 버튼, 이름 헤더 박스, 주요 강조 영역
- **사용 비율**: 전체 UI의 약 12%
- **팁**: 두꺼운 블랙 테두리(`#1A1A1A`)와 함께 사용하면 팝아트 느낌이 살아남

### Secondary — `#4BAE76` 미디엄 그린
- **사용 위치**: 섹션 헤더 박스 (About Me, Strengths & Interests 등)
- **사용 비율**: 전체 UI의 약 8%
- **팁**: 오렌지와 보색 관계로 함께 쓰면 생동감 있는 대비 효과

### Accent — `#F5C842` 골든 옐로우
- **사용 위치**: 보조 버튼(contact), 별 장식 아이콘, 포인트 요소
- **사용 비율**: 전체 UI의 약 3%
- **주의사항**: 화이트 텍스트와 함께 사용 시 대비율이 낮으므로 블랙 텍스트 사용 권장

### Background — `#FAF0E8` 웜 피치 크림
- **사용 위치**: 페이지 전체 배경
- **팁**: 순수 화이트보다 따뜻한 느낌을 주어 딱딱하지 않은 분위기 연출

### Border — `#1A1A1A` 다크 블랙
- **사용 위치**: 카드 외곽선, 버튼 테두리, 박스 outline (2~3px 두께 권장)
- **팁**: 두꺼운 테두리가 이 디자인의 핵심 — 얇게 쓰면 팝아트 느낌이 사라짐

### Text Colors
| 변수 | 색상 | 사용 상황 |
|------|------|-----------|
| `--color-text-primary` `#1A1A1A` | 거의 블랙 | 본문, 헤드라인, 강조 텍스트 |
| `--color-text-secondary` `#555555` | 미디엄 그레이 | 설명 텍스트, 서브 정보 |
| `--color-text-muted` `#888888` | 라이트 그레이 | 날짜, 메타정보, 플레이스홀더 |

### Extra Colors (개성 요소)
| 변수 | 색상 | 사용 상황 |
|------|------|-----------|
| `--color-profile-bg` `#A855C8` | 퍼플 | 프로필 사진 배경 |
| `--color-tech-stack` `#A0D0F0` | 라이트 블루 | Tech-Stack 버튼 |
| `--color-contact` `#F5C842` | 골드 옐로우 | Contact 버튼 |

## 반응형 고려사항

### 다크 모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1E1810;
    --color-bg-secondary: #2A2218;
    --color-bg-card: #2E2820;
    --color-text-primary: #F5EDE0;
    --color-text-secondary: #C8B8A8;
    --color-text-muted: #887870;
    --color-border: #F5EDE0;
    --color-border-light: #3A3028;
    /* 포인트 컬러는 유지 — 개성 보존 */
    --color-primary: #F4845F;
    --color-secondary: #4BAE76;
    --color-accent: #F5C842;
  }
}
```

### 모바일 접근성
- 본문 텍스트(`#1A1A1A`)와 화이트 카드 배경(`#FFFFFF`) 대비율: **약 19:1** — WCAG AAA 기준 충족
- 오렌지(`#F4845F`)와 화이트 배경 대비율: **약 2.9:1** — 버튼 내 텍스트는 반드시 블랙(`#1A1A1A`) 사용
- 옐로우(`#F5C842`) 버튼 위 텍스트는 블랙(`#1A1A1A`) 사용 필수
