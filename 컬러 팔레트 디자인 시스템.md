## 컬러 사용 가이드

### Primary — `#9B1F28` 딥 레드
- **사용 위치**: 메인 헤드라인, CTA 버튼, 강조 텍스트, 네비게이션 활성 상태
- **사용 비율**: 전체 UI의 약 8%
- **주의사항**: 배경에 단독으로 쓰지 않고 크림/화이트 배경 위에 텍스트나 버튼으로 사용

### Secondary — `#C2D4C4` 세이지 민트
- **사용 위치**: 페이지 전체 배경, 섹션 구분, 패턴 배경
- **사용 비율**: 전체 UI의 약 45%
- **팁**: 패턴이나 텍스처를 추가하면 원본 사이트의 스크랩북 느낌을 살릴 수 있음

### Accent — `#C4A04A` 웜 골드
- **사용 위치**: 호버 효과, 아이콘 포인트, 장식 요소, 뱃지
- **사용 비율**: 전체 UI의 약 5%
- **팁**: 남용하지 않고 특별한 강조 요소에만 제한적으로 사용

### Background — `#F5EFE0` 크림
- **사용 위치**: 카드, 콘텐츠 박스, 모달, 사이드바
- **팁**: 순수 흰색(#FFFFFF) 대신 사용하면 레트로 감성을 유지할 수 있음

### Text Colors
| 변수 | 색상 | 사용 상황 |
|------|------|-----------|
| `--color-text-primary` `#1E1E1E` | 거의 블랙 | 본문, 긴 텍스트 |
| `--color-text-secondary` `#5C4A3A` | 다크 브라운 | 서브타이틀, 캡션 |
| `--color-text-muted` `#8B7355` | 미디엄 브라운 | 날짜, 메타정보, 플레이스홀더 |

## 반응형 고려사항

### 다크 모드 대응
```css
@media (prefers-color-scheme: dark) {
  :root {
    --color-bg-primary: #1A1410;
    --color-bg-secondary: #2A2420;
    --color-bg-card: #2E2820;
    --color-text-primary: #F0EAD8;
    --color-text-secondary: #C4B89A;
    --color-text-muted: #8B7A65;
    --color-border: #4A3D30;
    /* Primary/Accent 계열은 유지 — 레트로 개성 보존 */
    --color-primary: #C4404A;
    --color-secondary: #3D5040;
  }
}
```

### 모바일 접근성
- 본문 텍스트(`#1E1E1E`)와 크림 배경(`#F5EFE0`) 대비율: **약 14:1** — WCAG AAA 기준 충족
- 딥 레드(`#9B1F28`)와 크림 배경 대비율: **약 7:1** — WCAG AA 기준 충족
- 작은 화면에서는 민트 배경 패턴을 단색(`#C2D4C4`)으로 단순화 권장
